Project Work

Here are a few key projects I developed to apply my learnings in data analytics, machine learning, and data engineering.

World Happiness Report 2024: Data Analysis

This project provides a comprehensive analysis of the World Happiness Report 2024. It examines global happiness trends, identifies key drivers, and uses regression modeling for predictive insights. The analysis reveals that Europe exhibits the highest median happiness scores, while Africa faces systemic challenges. A detailed regression model using factors such as GDP per capita, Social Support, and Healthy Life Expectancy achieved an R² of 0.77 with MAE of 0.40 and RMSE of 0.50. Visualizations include box-and-whisker plots, scatterplots, and heatmaps highlighting the top 10 happiest countries.

Fitbit Comments Sentiment Analysis using NLP

Leveraging NLP, this project analyzes over 900 customer reviews of Fitbit products. It involves web scraping using Selenium, extensive text preprocessing (cleaning, tokenization, lemmatization), and sentiment classification using TextBlob. Data visualizations (bar and pie charts) reveal trends in user feedback—highlighting both negative issues (like missing features and delayed notifications) and positive aspects such as accurate activity tracking and a user-friendly interface.

Soccer Match Win Prediction using AML

This project utilizes historical Premier League match data to predict match outcomes using Automated Machine Learning. Key features include categorical encoding of venues and opponents, extraction of time and day features, and rolling averages of match statistics. A RandomForestClassifier was trained on pre-2022 matches and tested on post-2022 data, achieving an accuracy of ~58% and an improved precision of 0.625 after including dynamic performance trends.

From Data To Glory: FIFA Player Analytics
